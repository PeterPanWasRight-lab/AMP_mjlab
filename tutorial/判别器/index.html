Adversarial Motion Priors: Discriminator Math & Dynamics

1. 判别器 D 是个什么？

判别器 D 就是一个打分函数，结构是 MLP（多层感知器）：

输入：一帧到下一帧的身体状态（13 个 body 的位置+姿态+速度，195 维 × 2 拼起来）

输出：一个标量 d，表示"像不像人"

核心：不看"姿势像不像"，看"动作的过渡像不像"——所以要 (s, s') 两个时刻

2. 判别器怎么训练？

LSGAN 损失（最小二乘 GAN）—— 让 D 给专家 +1 分，给 G1 -1 分：

expert_loss = MSE(D(专家的 (s,s')),  +1)   # 让 D 给专家高分
policy_loss = MSE(D(G1 的 (s,s')),   -1)   # 让 D 给 G1 低分
grad_pen    = λ · (||∇D||_2)²              # 让 D 学得"软"
d_loss      = 0.5 · (expert_loss + policy_loss) + grad_pen

把符号翻译成人话：

  • expert_loss = (D 对专家打的分 - 1)² —— D 看不到位的差距
  • policy_loss = (D 对 G1 打的分 + 1)² —— D 看不到位的差距
  • grad_pen = D 在输入附近的"陡峭程度"——惩罚 D 太尖锐

训练目的：让 D 看完数据后，对专家接近 +1，对 G1 接近 -1。

3. 风格奖励：style_reward = f(d)

训练好的 D 给 G1 打分 d 后，style_reward 这么算：

style_reward = clip(1 - 0.25 · (d - 1)²,  min=0)

7. 核心数学公式

7.1 LSGAN 损失

$$\mathcal{L}_D^{\text{expert}} = \frac{1}{2}\mathbb{E}_{(s_e, s_e') \sim \mathcal{B}_e}\left[\left(D(s_e, s_e') - 1\right)^2\right]$$

$$\mathcal{L}_D^{\text{policy}} = \frac{1}{2}\mathbb{E}_{(s_\pi, s_\pi') \sim \mathcal{B}_\pi}\left[\left(D(s_\pi, s_\pi') + 1\right)^2\right]$$

$$\mathcal{L}_D = \mathcal{L}_D^{\text{expert}} + \mathcal{L}_D^{\text{policy}} + \lambda \cdot \mathbb{E}_{x \sim \mathcal{B}_e}\left[\|\nabla_x D(x)\|_2^2\right]$$

7.2 风格奖励（推理时）

$$r_{\text{style}}(s, s') = w \cdot \max\left(0,\ 1 - \frac{1}{4}\big(D(s, s') - 1\big)^2\right)$$

其中 \(w = \text{amp\_reward\_coef}\)，\(\max(0, \cdot)\) 是 clip(min=0)。

7.3 奖励曲线推导

令 \(x = d - 1\)，则 \(x^2\) 是离 0 越远越大的平方函数。\(1 - 0.25 x^2\) 就是离 +1 越远越小的函数。

导数（梯度信号）：\(\frac{d}{dd} r_{\text{style}} = -0.5 (d - 1)\)

当 \(d = 0\) 时梯度 = 0.5（最大），当 \(d = -0.95\) 时梯度 = 0.025（很小）—— 这就是 D 太强时训练失败的原因。

8. 易混概念澄清

8.1 D 太强 vs D 过拟合

概念 描述的维度 问的问题
D 太强 D 当前的判别能力 "D 能不能完美区分专家和 G1"
D 过拟合 D 在训练集 vs 测试集上的差异 "D 是不是只记住了见过的样本"

两者症状相似（都崩训练），但根因不同：

  • D 太强：在所有数据上都好（包括 G1 没见过的）→ 评分太极端，梯度消失
  • D 过拟合：在训练集上好，新数据上差 → G 被训练集分布骗

在 AMP/GAN 里更准确的术语是 "discriminator dominance"（判别器支配）。

8.2 共性 vs 非共性的数学解释

判别器 D 学习的判别面在"风格方向"上很宽（步速 0.5~3.0 m/s 都在专家侧），但在"反风格方向"上很严。

原因：

  1. 梯度惩罚 grad_pen 让 D 在专家流形附近平滑（沿流形方向 D 不变）
  2. 多样数据 让 D 找不到"在所有专家中都成立"的非共性特征
  3. LSGAN 损失 强制 D 找最强分类特征 → 只有共性满足要求

9. 核心要点总结

  • AMP = PPO + 风格奖励。判别器只算"像不像人"的分数，这个分数加到 env reward 上，PPO 该怎么训就怎么训。
  • D 是 70-80% 准确率是"G1 能感受到改进信号"的最佳状态。grad_pen 让 D 自然不会冲到 100%。
  • 多样性数据让 D 只能学到"风格共性"，剥离步速/方向/个体差异等"非共性噪声"。
  • 没法设计完美奖励——因为"什么是像人"本身就没法用人手定义。D 存在就是为了让神经网络自己发现这个标准。
  • D 强 ≠ D 过拟合。前者是判别能力太强，后者是泛化能力差。两者症状相似但根因不同。
